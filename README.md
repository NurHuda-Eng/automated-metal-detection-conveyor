# Automated Metal Detection Conveyor System

## Project Overview

This project is a PLC-based automated conveyor system designed to automatically detect and sort metal objects using industrial sensors and pneumatic actuators.

## Technologies Used

- Mitsubishi FX3U PLC
- GX Works2
- Proximity Sensor
- Pneumatic Cylinder
- Solenoid Valve
- 24VDC Power Supply

## Main Features

- Automatic metal object detection
- PLC-based control logic
- Pneumatic sorting mechanism
- Conveyor automation system
